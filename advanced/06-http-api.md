# HTTP API 接口服务

## koa2 代码生成器

要求：

- 检查 koa2 代码生成器软件是否安装，`koa2 --version`
- 安装 koa2 代码生成器软件，`npm install -g koa-generator`，如果已经安装跳过此步
- 用 koa2 代码生成器创建项目框架代码，`koa2 -e 06-app`
- 安装项目依赖，`cd 06-app && npm install`
- 运行程序，`DEBUG=06-app:* npm start`
- 用浏览器访问这个网站
- 查看框架代码的目录结构，搞明白每个文件夹的作用
- 查看阅读生成的代码文件
