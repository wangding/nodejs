- [zlib API 资料](http://nodejs.cn/api/zlib.html)  

## GZip 压缩文件

要求：  
- 创建 20-zlib 目录  
- 编写 01-my-gzip.js 脚本
- 要求支持命令行参数，包括：要压缩的文件名
- 没有命令行参数时，提示错误信息
- 文件不存在时，提示错误信息
- 文件存在，则用 GZip 算法压缩该文件
- 打印报告，压缩前文件的字节数，压缩后文件的字节数
